# 关于 FastD 

FastD 是一个为构建高性能 API 而生的框架，让开发者更加专注应用。

FastD 提供灵活配置的核心，由于为了性能，提供的组件当不能满足当下业务时，可以通过自定义对核心，和服务进行扩展。

由于 FastD 是支持 Swoole 扩展，因此可以通过 Swoole 来开发更多的应用，打破传统web模式，还有很多想象空间。

#### 适用场景

* 适用于专注 API 端的开发

#### 环境要求

* PHP >= 5.6
* ext-curl
* ext-pdo
* [ext-swoole]
